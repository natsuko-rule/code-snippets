Web技術 用語集

HTML
HyperText Markup Languageの略。Webページの構造を定義するためのマークアップ言語。
CSS
Cascading Style Sheetsの略。Webページの見た目やレイアウト（スタイル）を指定するための言語。
JavaScript
Webページに動きやインタラクティブな機能を追加するためのプログラミング言語。ブラウザ上で動作する。
DOM
Document Object Modelの略。HTMLやXML文書をプログラムから操作するためのインターフェース（API）。JavaScriptはこのDOMを通じてHTML要素を変更したりする。
API
Application Programming Interfaceの略。ソフトウェアコンポーネント同士が互いに情報をやり取りする際に使用されるインターフェース（接続仕様）のこと。

Q&A形式の例

Q. HTMLとは何ですか？
A. Webページの骨組みを作るための言語です。
Q. CSSの役割は何ですか？
A. Webページの見た目を装飾する役割を持ちます。
